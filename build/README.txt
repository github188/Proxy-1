
[以下名称最后的问号"?" 表示一个数字]

## 配置文件内容说明
1.系统配置 [SYSTEM] 
	配置的是一般系统属性，例如是否启用monitor等
	[SYSTEM]下具体配置：
	CacheLiveSecs :   需要转发的数据最大缓存秒数
	EnableBackup :    是否启用备份
	EnableMonitor:    看门狗开关
	cvtrule? :        每一组转换所使用的规则
2. 配置一个转发组：
    每一个转发组以成对的 [LEFT?] [RIGHT?]配置
	[LEFT] [RIGHT]分别表示该转发组的两边的连接
  
3. 配置连接：
   每一个[LEFT?] [RIGHT?]下面可以配置多个连接
   每一个连接包含一个mod? 一个ip? 一个port?

## 配置文件配置值对照表
mod?=
	0 : 客户端模式，对应的ip?=需要给出服务端的IP
	1 : 服务器模式，对应的ip?=现在未使用，可以填0
